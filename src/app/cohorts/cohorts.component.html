How to create new Cohorts using OHDSI Atlas

To create patient cohorts please use OHDSI's Tool Atlas.
You can create cohorts just like you are used to. The only thing to remember is the ID of the cohort created to retrieve all the patients IDs the cohort includes.
There are two ways to determine a cohort's ID:

  • Look at the column "Id" in the Cohort Summary View.
  • Look at a concrete Cohort's URL: e.g. http://localhost:8081/atlas/#/cohortdefinition/3.
